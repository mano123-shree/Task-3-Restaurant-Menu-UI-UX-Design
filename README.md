This repository contains **Task 3** of the internship assignment.

The task focuses on designing a **restaurant menu UI/UX** that is visually appealing, easy to use, and well categorized to help customers quickly find dishes and encourage them to order more.

## Task Overview
A restaurant menu UI/UX project involves designing a digital menu that improves customer experience by combining clear layout, attractive visuals, and smooth navigation.

## Key Design Principles
- **Visually Appealing:**  
  The menu uses carefully selected colors, fonts, and images to create an attractive and readable interface.

- **Easy to Use:**  
  The menu layout is simple and intuitive, allowing customers to browse items and make selections effortlessly.

- **Categorized:**  
  Menu items are organized into clear and consistent categories, helping users quickly find the dishes they are looking for.
